Description

StV (structural HOR variant) annotation of alpha satellite higher-order repeats in T2T CHM13 human genome assembly release v1.0.

Methods

The StV track, (structural HOR variant, as opposed to sequence variant (SqV), usually can be shown as whole-monomer in/del of the full-size (canonical) HOR) was derived from the AS HOR annotation track (HOR-track; AS_HOR_Annot.bed). The main idea of generating the StV map was iterating through AS HOR annotation bed file and cutting after the last monomer in a HOR (the one with maximal number in a given array) or before the first monomer (the one with number #1) if the last monomer is not present in a given StV.

See details: Altemose et al., 2021, Supplementary Materials and Methods, Section 6: HOR unit and NRS prediction. HOR Structrual Variant (StV) prediction using CHM13 hor-monomer annotation.
